DocAgent

Date August 2025
Type Agentic RAG System
Status Completed

An advanced agentic document QA system that leverages Retrieval-Augmented Generation (RAG) with Model Context Protocol (MCP) to create intelligent agents capable of parsing, embedding, retrieving, and answering queries from multiple document formats.

Architecture & Design

DocAgent employs a sophisticated multi-agent architecture where specialized agents handle different aspects of document processing:

  • Ingestion Agent: Handles document parsing and pre-processing
  • Retrieval Agent: Performs semantic search using FAISS
  • LLM Response Agent: Generates responses using Llama 3.1 8b
  • Coordinator Agent: Orchestrates communication between agents

Tools & Tech Stack

Built with LangChain, HuggingFace Inference Client, Llama 3.1 8b, FAISS, and Gradio. The system leverages Retrieval-Augmented Generation (RAG) with a Model Context Protocol (MCP) for coordinated multi-format document QA.

Tamil Speech-to-Text ASR

Date March 2025
Type Automatic Speech Recognition
Status Completed

Fine-tuned OpenAI's Whisper model for Tamil automatic speech recognition using a custom dataset of over 80 hours own conversational Tamil audio dataset.

Key Achievements

  • Achieved Word Error Rate (WER) of 22
  • Preprocessed and augmented dataset for accent diversity
  • Used Hugging Face for training and versioning

Training and evaluation data

The model was fine-tuned on a custom web-scraped dataset called "TamilVoiceCorpus" (available at ragunath123/TamilVoiceCorpus on Hugging Face). This dataset consists of Tamil speech audio paired with accurate transcriptions. For training, 12,000 samples were used from the dataset, while 3,000 samples were used for evaluation. The audio was processed by resampling from 48kHz to 16kHz to match Whisper's requirements. The dataset includes a diverse range of Tamil speech samples, which helps the model generalize across different speakers, accents, and content types.

TamilVoiceCorpus

Date March 2025
Type Dataset-Automatic Speech Recognition
Status Completed

TamilVoiceCorpus is an open-source Tamil speech dataset, and built from publicly available sources. It is intended for use in training and evaluating automatic speech recognition (ASR) systems and other speech processing models for the Tamil language.

Dataset Structure

• PureVox

The PureVox subset contains cleaner audio with minimal background noise, making it ideal for baseline ASR model training and evaluation.

• AmbientVox

The AmbientVox subset includes audio with background noise, simulating real-world conditions for more robust ASR model training.

Info

This is a dataset for Automatic Speech Recognition (ASR) focused on conversational Tamil, collected from various public sources on the web. Each sample is a short audio clip (averaging 10 seconds) paired with its corresponding transcription.

Chain-of-Thought(Math)-Dataset

Date May 2025
Type Dataset-Math Reasoning(Chain of Thought)
Status Completed

This dataset is a preprocessed and filtered subset of the NVIDIA Open Math Reasoning Dataset, focused on the Chain-of-Thought (CoT) reasoning split. It has been adapted for compatibility with large language model (LLM) fine-tuning, such as LLaMA and similar architectures. Each sample includes a math problem, detailed step-by-step reasoning, and the final answer — all structured as a single natural language text field, making it plug-and-play for autoregressive training.

Dataset Summary

  • Split: Chain of Thought (CoT) only
  • Total Rows: 3,201,061
  • Data Format: Plain text, LLM-tokenizer friendly
  • Primary Use Case: Training and evaluating LLMs on complex mathematical reasoning tasks using chain-of-thought prompting

Source

  • Original Dataset: NVIDIA Open Math Reasoning
  • This Version: Chain-of-Thought split, extracted and preprocessed for LLM use (cleaned, formatted, deduplicated where necessary)

Space Debris Tracking

Date October 2024
Type Object detection and Tracking
Status Completed

This project demonstrates a space debris tracking system using YOLOv5 for object detection and a Kalman Filter to predict and track the debris's trajectory in real time.

Project Overview

The system processes video frames to detect space debris, predict its future position, and visualize the trajectory. It uses YOLOv5 for real-time object detection and a Kalman Filter for trajectory prediction.

Features

  • Real-Time Detection: YOLOv5 detects debris in video frames. Kalman Filter
  • Prediction: Predicts the debris's future position based on previous frame positions.
  • Visual Tracking: Draws bounding boxes and predicted trajectory on each frame.
  • Video Output: Saves a new video file with tracked debris and predictions.

Histopathology Image Captioning

Date July 2025
Type Image Captioning
Status Completed

This model is a fine-tuned version of Salesforce/blip-image-captioning-base on a histopathology image dataset, trained to automatically generate captions for digital pathology slides. It achieved an average loss of 0.0098 on the training set.

Model Training & Evaluation

Trained the BLIP model on a high-performance GPU cluster with mixed precision, using a custom dataset of histopathology images.

Training Details

  • Dataset: Histopathology images with corresponding captions
  • Epochs: 10
  • Base Model: Salesforce/blip-image-captioning-base
  • Batch Size: 4
  • Learning Rate: 5e-5
  • Gradient Accumulation: 2
